# 隱私權政策 (Privacy Policy)

**最後更新日期：2026 年 6 月 23 日**

## 1. 概述

本隱私權政策說明 **LiuLianBot**（以下簡稱「本機器人」或「我們」）如何收集、使用、儲存及保護您的資料。我們重視您的隱私權，並致力於透明地揭露我們的資料處理方式。

## 2. 我們收集的資料

### 2.1 自動收集的資料

當本機器人被加入 Discord 伺服器後，會自動收集以下資料以提供核心功能：

| 資料類別 | 具體內容 | 用途 |
|---------|---------|------|
| **伺服器識別資訊** | Discord 伺服器 ID（Guild ID）、頻道 ID（Channel ID） | 設定儲存、記錄頻道對應、私人語音頻道管理 |
| **使用者識別資訊** | Discord 使用者 ID（User ID） | 權限管理、私人語音頻道擁有權識別 |
| **訊息內容** | 被編輯訊息的原始及修改後內容、被刪除訊息的內容、附件檔案名稱及 URL | 伺服器事件記錄功能 |
| **語音狀態** | 使用者加入/離開/移動語音頻道的記錄 | 語音事件記錄功能 |
| **成員事件** | 成員加入/離開時間、暱稱變更、身份組變更、封鎖/解封記錄 | 成員事件記錄功能 |
| **伺服器事件** | 頻道建立/刪除/修改、身份組建立/刪除/修改、伺服器設定變更 | 伺服器事件記錄功能 |

### 2.2 使用者主動提供的資料

| 資料類別 | 具體內容 | 用途 |
|---------|---------|------|
| **語言偏好設定** | 選擇的語言（英文或繁體中文） | 提供多語言介面 |
| **私人語音頻道設定** | 自訂頻道名稱、人數上限、權限設定 | 私人語音頻道個人化 |
| **抽選設定** | 抽選頻道指定、結果發送方式（私訊/頻道） | 抽選功能設定 |

### 2.3 我們不收集的資料

我們明確**不**收集以下資料：
- 電子郵件地址
- 真實姓名或身分證件
- IP 位址
- 瀏覽器或裝置資訊
- 語音通話內容或音訊
- 信用卡或付款資訊（本機器人為完全免費的服務）

### 2.4 敏感資料警告

⚠️ **訊息內容記錄**：當伺服器管理員啟用記錄功能時，本機器人會將被編輯及被刪除的訊息內容（包含文字及附件資訊）記錄到指定的 Discord 文字頻道中。這些記錄會包含訊息作者的 Discord 顯示名稱、頭像 URL 及使用者 ID。**伺服器管理員有責任告知其伺服器成員此記錄功能的存在。**

## 3. 資料的使用方式

我們收集的資料僅用於以下目的：

- **提供核心功能**：執行指令、管理私人語音頻道、產生隨機抽選結果。
- **伺服器記錄**：按伺服器管理員的設定記錄伺服器事件。
- **權限管理**：驗證使用者權限等級以控制指令存取。
- **語言在地化**：根據伺服器設定的語言提供對應的介面文字。
- **除錯與改進**：分析錯誤記錄以改善機器人穩定性。

我們**不會**：
- 將資料用於廣告或行銷目的。
- 建立使用者行為檔案或進行分析。
- 將資料出售或分享給第三方。
- 使用資料訓練人工智慧模型。

## 4. 資料儲存

### 4.1 儲存位置與方式

| 資料類型 | 儲存方式 | 儲存位置 |
|---------|---------|---------|
| 伺服器/頻道/使用者 ID、語言設定、抽選設定 | MySQL/MariaDB 資料庫 | 機器人營運者管理的伺服器 |
| 私人語音頻道設定（含 config_json） | MySQL/MariaDB 資料庫 | 機器人營運者管理的伺服器 |
| 機器人設定（管理員清單、Token 等） | `config.json` 檔案 | 機器人執行環境的本機磁碟 |
| 事件記錄（訊息、語音、成員等） | Discord 文字頻道中的 Embed 訊息 | Discord 伺服器 |
| 執行記錄（指令使用記錄、錯誤記錄） | `logs/` 目錄中的日誌檔案 | 機器人執行環境的本機磁碟 |

### 4.2 資料保留政策

- **私人語音頻道設定**：自動在 30 天後清除未使用的設定。
- **資料庫記錄**：除非伺服器管理員移除機器人或重設設定，否則將持續保留。
- **Discord 記錄頻道訊息**：由 Discord 平台及伺服器管理員的頻道管理政策管轄。
- **日誌檔案**：以日期命名（`bot_YYYYMMDD.log`），由機器人營運者手動管理。

## 5. 第三方服務

本機器人使用以下第三方服務：

| 服務 | 用途 | 傳送的資料 |
|------|------|----------|
| **Discord API** | 機器人運作、訊息傳送、事件接收 | 所有機器人功能所需的 Discord 資料 |
| **MySQL/MariaDB** | 持久化資料儲存 | 伺服器 ID、頻道 ID、使用者 ID、設定資料 |
| **Ubisoft 官方網站** | 爬取 R6 地圖與幹員資料（`mapsgrap.py`） | 無（僅讀取公開頁面） |
| **GitHub** | 程式碼更新功能（`>update` 指令） | 無使用者資料 |

除上述服務外，本機器人**不會**將資料傳送給任何分析服務、廣告網路、CDN 或其他第三方。

## 6. 使用者權利

### 6.1 您的權利

作為使用者，您擁有以下權利：

- **查閱權**：您可以要求查閱我們所持有的與您相關的資料（您的 Discord 使用者 ID 相關的設定）。
- **刪除權**：您可以要求刪除我們所儲存與您相關的資料。
- **拒絕記錄權**：您可以請求伺服器管理員關閉訊息/事件記錄功能，或將特定頻道排除在記錄範圍之外。
- **退出權**：您可以隨時透過離開使用本機器人的伺服器來停止與本機器人的互動。

### 6.2 權利的行使

若要行使上述權利，請透過 GitHub Issues 或 Discord 聯絡機器人開發者。我們將在合理的時間內回應您的請求。

⚠️ **請注意**：某些資料（如已發送至 Discord 記錄頻道的訊息記錄）一旦發送後，可能無法從 Discord 平台上完全移除。這類資料的刪除需遵循 Discord 的資料保留政策。

## 7. 資料安全

我們採取適當的技術措施來保護您的資料：

- 資料庫連線使用認證機制（使用者名稱/密碼）。
- 機器人設定檔（`config.json`）儲存於伺服器本機，不對外公開。
- 僅機器人營運者擁有伺服器的存取權限。

然而，**沒有任何網路傳輸或電子儲存方式是 100% 安全的**。雖然我們努力保護您的資料，但我們無法保證其絕對安全。

## 8. 兒童隱私

本機器人不會故意收集 13 歲以下兒童的個人資料。如果您是家長或監護人，且發現您的孩子向我們提供了個人資料，請與我們聯絡，我們將立即刪除該資料。

## 9. 國際資料傳輸

本機器人的伺服器可能位於台灣或其他國家/地區。您的資料可能會在您所在國家/地區以外的伺服器上儲存和處理。使用本機器人即表示您同意此類資料傳輸。

## 10. 隱私權政策變更

我們可能會不定期更新本隱私權政策。重大變更將透過以下方式通知：

- 更新本文件頂部的「最後更新日期」。
- 在 GitHub 儲存庫中記錄變更。
- （如適用）透過機器人的公告功能通知伺服器管理員。

繼續使用本機器人即表示您接受更新後的隱私權政策。
